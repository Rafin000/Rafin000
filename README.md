## Projects I am proud of

**🤖 Autonomous Coding Agent**
> A TypeScript coding agent that learns microservice repos using Neo4j graph knowledge and Cloudflare Vectorize embeddings. Implements features via Claude with streaming tool use and a stateful plan, review, implement and sync workflow. Connected to GitHub PR webhooks and Jira automation.

**🧠 Multi-Channel AI Customer Service Platform**
> Multi-tenant SaaS that deploys AI agents across WhatsApp, Telegram, Facebook, Instagram and web widgets. Built with RAG knowledge base, workflow builder, Stripe billing and real-time messaging using Claude and OpenAI.

**🎙️ Real-Time AI Voice System** *(at Forward Flow)*
> An AI assistant that joins live phone calls and responds intelligently in real time using Twilio, ElevenLabs and OpenAI with sub-second latency.

**🔍 Offensive Bengali Meme Detection** *(Research / Thesis)*
> Multimodal weighted ensemble combining BERT and DenseNet to detect offensive Bengali memes across a three level classification hierarchy. Won Champion at CLBLP 2023.

**🔐 OTP Detection with NLP**
> Robust OTP detection system using Python, spaCy, TF-IDF and PostgreSQL for multilingual SMS filtering with dynamic rule management.

**🏥 Medicare Backend**
> Node.js backend using Express, Prisma and MySQL to find the nearest doctor based on patient needs and location.
