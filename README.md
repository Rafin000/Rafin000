## Projects I am proud of
 
**Autonomous Coding Agent**
> A TypeScript coding agent that learns microservice repos using Neo4j graph knowledge and Cloudflare Vectorize embeddings. Implements features via Claude with streaming tool use and connects to GitHub PRs and Jira automatically.
 
**Multi-Channel AI Customer Service Platform**
> Multi-tenant SaaS that deploys AI agents across WhatsApp, Telegram, Facebook, Instagram and web widgets. Built with RAG knowledge base, workflow builder and real-time messaging using Claude and OpenAI.
 
**Real-Time AI Voice System** *(at Forward Flow)*
> An AI assistant that joins live phone calls and responds intelligently in real time using Twilio, ElevenLabs and OpenAI with sub-second latency.
 
**Offensive Bengali Meme Detection** *(Research / Thesis)*
> Multimodal weighted ensemble combining BERT and DenseNet to detect offensive Bengali memes. Won Champion at CLBLP 2023 international workshop.
 
**OTP Detection with NLP**
> Robust OTP detection system using Python, spaCy, TF-IDF and PostgreSQL for multilingual SMS filtering.
